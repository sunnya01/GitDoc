SAP财务会计凭证屏幕配置

主要由OB41针对记账码的控制 和 OBC4 -> OBC5 ->FS00的总账科目控制

当你做一笔凭证的时候

首先会输入记账码 ，记账码会通过OB41(TBSL)找寻对应的唯一配置

然后如果是应收(应付)的记账码,系统会先根据填的客商查看客商该公司视图下是否存在对应的统驭科目,如果不存在

则取公司代码在FS00配置的公司层级的统驭科目-A,OBC5下配置了该公司代码对应的OBC4分组，假设是Z001分组

双击分组进入后可以看到里面有许多的字段状态组,这个字段状态组关联的是总账科目的类型，当你输入一个客商后

通过上面逻辑取到的统驭科目在FS00中与当前公司代码会有一个组合配置，其中有一项配置就是配置当前公司+该科目对应的

字段状态组。OBC4双击字段状态组可以看到内部的与OB41一样的屏幕配置，两者取与的逻辑运算

如果输入的是总账的记账码，则直接那总账科目进行上述逻辑判断
