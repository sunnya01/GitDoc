## 计算机指令

#### Windows指令: 

cd 进入 

cd..返回上层 

cd/返回盘目录 

echo word>filename.doc 创建文件 

dir 输出目录下文件清单 

md 新建目录 

rd 删除目录 

del filename.doc删除文件 也可以输入目录名称 实际删除目录下所有文件

#### LINUX指令:

###### 1. ls：列出当前目录中的文件和子目录

```text
ls
```

###### 2. pwd：显示当前工作目录的路径

```text
pwd
```

###### 3. cd：切换工作目录

```text
cd /path/to/directory
```

###### 4. mkdir：创建新目录

```text
mkdir directory_name
```

###### 5. rmdir：删除空目录

```text
rmdir directory_name
```

###### 6. rm：删除文件或目录

```text
rm file_name
rm -r directory_name  # 递归删除目录及其内容
```

###### 7. cp：复制文件或目录

```text
cp source_file destination
cp -r source_directory destination  # 递归复制目录及其内容
```

###### 8. mv：移动或重命名文件或目录

```text
mv old_name new_name
```

###### 9. touch：创建空文件或更新文件的时间戳

```text
touch file_name
```

###### 10. cat：连接和显示文件内容

```text
cat file_name
```

###### 11. more/less：逐页显示文本文件内容

```text
more file_name
less file_name
```

###### 12. head/tail：显示文件的前几行或后几行

```text
head -n 10 file_name  # 显示文件的前10行
tail -n 20 file_name  # 显示文件的后20行
```

###### 13. grep：在文件中搜索指定文本

```text
grep search_term file_name
```

###### 14. ps：显示当前运行的进程

```text
ps aux
```

###### 15. kill：终止进程

```text
kill process_id
```

###### 16. ifconfig/ip：查看和配置网络接口信息

```text
ifconfig
ip addr show
```

###### 17. ping：测试与主机的连通性

```text
ping host_name_or_ip
```

18. ###### vim: 按照字符编辑文件

19. ###### touch: 新建文件
