## 日期

#### LocalDateTime(本地日期时间)

<u>*now*()</u>获取当前的本地时间，本地时区的一个LocalDateTime实例对象，可选参数ZoneId可以设置时区

<u>*of*(int year, int month, int dayOfMonth, int hour, int minute)</u>可以传入自定义日期获取LocalDateTime实例对象

<u>*ofInstant*(Instant instant, ZoneId zone)</u>将Instant转换为LocalDateTime实例对象,由于Instant只保存自1970年01月01日以来的毫秒数，因此必须附带时区属性才能转换为本地时间对象

*<u>minus****||plus****</u>*对日期进行按照****的加减(****可以是Days、Hours、Weeks、Months、Years)

*<u>with****</u>*对日期进行设定～～

*<u>parse(CharSequence text)</u>*通过字符串解析得到LocalDateTime实例对象,text可以是String实例对象因为String实现了CharSequence接口"2007-12-03T10：15：30"

#### ZonedDateTime(带时区属性的时间)

<u>*of(LocalDateTime localDateTime, ZoneId zone)*</u>将LocalDateTime结合传入的ZoneId对象转换为含时区属性的ZonedDateTime实例对象

*<u>其他方法与LocalDateTime一致不再赘述</u>*

#### ZoneId(默认时区主体类)

<u>*getAvailableZoneIds*()</u>获取所有可用的ZoneId的String对象集合

<u>*of*(String zoneId)</u>根据上面的系统存在的ZoneId的String对象生成一个ZoneId实例对象

*ISO_LOCAL_DATE*<u>*systemDefault*()</u>获取一个系统默认的ZoneId实例对象

#### Instant(瞬时类默认存储从1970/01/01至今的毫秒数epochMilli)与util.Date类似

<u>System.*currentTimeMillis*()</u>系统默认静态方法 获取一个long型数据的epochMilli

<u>*ofEpochMilli(long epochMilli)*</u>通过epochMilli得到一个Instant实例对象

<u>*parse(final CharSequence text)*</u>通过解析描述了epochMilli的String字符串得到一个Instant实例对象

*<u>minus****||plus****</u>*对时间戳epochMilli进行按照****的加减(****可以是秒、毫秒、纳秒)

*<u>atOffset(ZoneOffset.ofHours(int hours))</u>*进行时间戳的偏移 结合 **<u>ZoneOffset*</u>**(time包下的时区转换工具)*的静态方法使用

***!!!***Instan保存的时间精确到纳秒级别,且永远是基于元时间1970.01.01开始计算的,因此转换到其他日期时间对象需要结合时区

#### DateTimeFormatter(日期格式转换工具类)

*<u>DateTimeFormatter</u>*包含了很多默认的静态的工具类实例对象(例:DateTimeFormatter.*ISO_LOCAL_DATE*)

*<u>ofPattern(String pattern)</u>*通过自定义模版生成一个DateTimeFormatter转换工具实例对象(例:yyyy-MM-dd)

*<u>ofLocalizedDate(FormatStyle dateStyle)</u>*结合FormatStyle实例对象获得一个可以根据ZonedDateTime和Locale动态转换时区输出和动态根据地区展示不同格式的格式化器(FormatStyle、Locale的实例对象均保存在本类的静态属性中)

*<u>format(TemporalAccessor temporal)</u>*DateTimeFormatter的实例方法,实现日期格式化的核心,通过传入的TemporalAccessor接口的实现类例如ZonedDateTime、LocalDateTime等进行格式化并返回一个符合格式要求的String对象

<u>*parse(CharSequence text)*</u>按照实例对象创建时的格式解析一个字符串并返回一个TemporalAccessor接口

Duration 计算时间间隔基于时间戳计算

Period计算日期间隔基于日期时间对象计算betwwen

TemporalAdjusters可以根据静态方法返回一个可以根据日历调整日期的调整器对象

Clock一个可以实时读取指定时区的时间戳的类
